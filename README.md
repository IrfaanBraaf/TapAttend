# TapAttend – NFC Attendance Tracking System

TapAttend is a mobile and web-based attendance tracking system that uses NFC technology to record employee or student check-ins and check-outs. The system provides a fast, secure, and automated alternative to traditional attendance methods.

## Features

- NFC tag scanning for attendance
- Real-time check-in and check-out tracking
- Admin dashboard for attendance monitoring
- Attendance history and reporting
- User profile management
- Secure authentication system
- Export attendance reports

## Tech Stack

Mobile Application
- React Native
- React Navigation
- NFC Manager

Web Dashboard
- React
- Material UI

Backend
- Node.js / ASP.NET Core API

Database
- Firebase Realtime Database 

Other Tools
- REST APIs
- Cloud storage for user profiles

## How It Works

1. Each user is assigned an NFC card or tag.
2. The mobile application scans the NFC tag.
3. The system records the timestamp and user ID.
4. Attendance data is stored in the database.
5. Administrators can view reports and attendance analytics from the dashboard.
